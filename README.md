git init - инициализация

**git commit -m "*first commit*"** - фиксация изменений

**git branch -M *main*** - переименование ветки

**git remote add origin *https://github.com/Andreyanchev-Leonid7/1.git*** - добавление к локальному репозиторию

**git push -u origin *main*** - публикуем

**mkdir *c1-hw-1*** - создание папки

**cd *c1-hw-1*** - переход в папку в Git

**touch *README.mn*** - создание файла

**ls** - статус
